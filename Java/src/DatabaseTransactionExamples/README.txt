INSTRUCTIONS

Before running any example in the package se.miun.itm.input.sql.example.query,
transfer the necessary experimental descriptors to the database by running the class
se.miun.itm.input.sql.example.transfer.ExperimentArchivesToDatabaseTransferrer

Before running any example in any of the other packages, transfer the
necessary experimental descriptors to the database by running the class
se.miun.itm.input.sql.example.transfer.XMLFilesToDatabaseTransferrer

NOTE: The examples has only been tested with the DBMS PostgreSQL 9.2,
and may therefore not work with other systems/versions.
